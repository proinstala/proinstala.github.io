Compati

Proyecto de Control y Automatización del Alumbrado.

Este proyecto es una combinación entre programación, electrónica y electricidad. diseñado para proporcionar un control eficiente y automatizado del alumbrado en el patio de la casa de mi madre.

Descripción del Proyecto

El objetivo principal de este proyecto es automatizar el sistema de iluminación del patio, permitiendo un control flexible y eficiente de las luces mediante el uso de tecnología Arduino y componentes electrónicos adicionales. El sistema se basa en una placa Arduino mega, para el control general y una placa arduino uno para el control de la pantalla tactil que proporciana una interfaz de usuario para interactuar con el sistema. Tambien mencionar que el arduino mega tiene una placa electronica acoplada (escudo) y una placa electronica de expansión. Todos los componentes estan integrados en un cuadro electrico de superficie instalado en el patio para tener el control del lugar. En el frontal del cuadro, se encuentra la pantalla tactil y cuatro pulsadores de accionamiento del encendido de las zonas principales.

La Pantalla tactil es la interfaz de usuario principal para poder interactuar con Compati. En esta foto se puede ver el menu principal donde marca la hora, la temperatura y los botones principales.

El armario electrico esta hecho de plastico ABS. Su tamaño es de 30cm de ancho por 40 de alto. En su interior se encuentran todos los componentes electricos y electronicos para el control de la instalación.

A la izquierda, el ardunino principal con un escudo acoplado encima y a la derecha la placa uxiliar de control con los driver de salidas y entradas, el modulo de reloj y el modulo Bluetooth.

El armario cuenta con interruptor magnetotermico de 10Amp y fuente de alimentacion con salida regulada a 9 Vdc.

En la puerta del armario se encuentra mecanizados los pulsadores de control de las zonas principales y la pantalla tactil.

La temperatura exterior es medidia con el sensor DS18B20 y para medir el nivel de luz exterior se usa fotoresistencia LDR mecanizada en una caja de conexión.
